This is an index for the Kashmiri-English Dictionary for second-language learners by Koul, Raina, & Bhat. It is intended to help you locate entries in the dictionary. given that the document is not searchable, itself.

You can use your browser search on the List tab to find a term or a character or a character sequence: search on the romanisation in the dictionary, or on the IPA rendition. (Note that the IPA has been generated automatically, so don't take it as perfect.) Once you find the term, scroll upwards and you'll see what page that term appears on.

You can also search for items in the Find tab. This is much more powerful because it allows you to use regular expression syntax for your search (see the Help tab). It is particularly useful for finding all entries that contain a given character or characters. It won't show you the page number, but you can find that by searching on the List tab.
