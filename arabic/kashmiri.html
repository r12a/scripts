This page gathers together basic information about the Arabic script and its use for the Kashmiri language. It aims (generally) to provide an overview of the orthography and typographic features, and (specifically) to advise how to write Kashmiri using Unicode; for greater details follow the footnote links (especially those with an arrow alongside them).

See the Arabic page for most of the information about how the Arabic script works, and the orthography used for the Arabic language. This page aims to provide Kashmiri-specific information.

For character-specific details follow the links to the Arabic character notes. See also the Kashmiri (arab) character app, and the notes on Kashmiri (deva).

For similar information related to this and other scripts, see the script links pages.

Clicking on red text examples, or highlighting part of the sample text shows a list of characters. Click on the vertical blue bar (bottom right) to change font settings for the sample text. Colours and annotations on panels listing characters are relevant to their use for the Kashmiri language.

Unless in parentheses, the transcriptions in italics that follow Kashmiri text are a transliteration developed for these pages. Those in parentheses are usually more standard transcriptions. Transcriptions in ⌈ brackets ⌋ may be phonemic or phonetic.

Usage & history

For information about the script in general, see the Arabic overview.

Kashmiri is written in the devanagari script by Hindus. Muslims use the arabic script.

See the comparison table

Distinctive features

The Arabic script is an abjad. This means that in normal use the script represents only consonant and long vowel sounds. The Arabic alphabet has been adapted to cope with the many more vowels sounds in languages such as Kashmiri. See the table to the right for a brief overview of features, taken from the Script Comparison Table.

Kashmiri is principally written using the nasta'liq style of Arabic writing. Glyphs are more drawn out, and the baseline tends to be sloping from word to word.

Text direction

Text is normally written horizontally, right to left, however numbers and non-Arabic script text runs left to right.

See the Arabic overview for more details, especially related to sequences of items and numbers.

Vowel details

The allocation of characters to vowel sounds is somewhat complicated. The complexity arises from the number of vowels in Kashmiri compared to the Arabic language, and the need to represent them all, but also because different sequences are needed for different positional forms. Also there are often a number of alternative character sequences possible.

Vowels in initial or isolated position are written with a preceding ا [U+0627 ARABIC LETTER ALEF].

The set of characters needed to represent the Kashmiri vowels, then, including decompositions and compositions, includes the following. The list may need to be adjusted when the representation of e is better understood.

The Unicode character ٳ [U+0673 ARABIC LETTER ALEF WITH WAVY HAMZA BELOW] represents a valid visual arrangement for Kashmiri, but is deprecated by the Standard, in favour of the decomposed sequence ا + ٟ [U+0627 ARABIC LETTER ALEF + U+065F ARABIC WAVY HAMZA BELOW], because there is no normalisation equivalence.

Vowel absence

Kashmiri uses ْ [U+0652 ARABIC SUKUN] to kill the vowel after a consonant.l2 Note that the shape may resemble an inverted v. This is also called jazm.

Sometimes people use ۡ [U+06E1 ARABIC SMALL HIGH DOTLESS HEAD OF KHAH] for this.

LOC has an example, روْنگ rw°ng, where the vowel absence is indicated over the WAWand not over the NOON.

Nasalisation

It seems that ں [U+06BA ARABIC LETTER NOON GHUNNA] may be used for nasalisation of a vowels. To be confirmed. See details about how this works in Urdu.

Basic consonants

The following constitute a basic set of consonants used for Kashmiri, that cover all standard phonemes.

Aspirated consonants

Six additional letters of the alphabet represent aspirated sounds. These are all written by combining a standard character with a following ھ [U+06BE ARABIC LETTER HEH DOACHASHMEE].

Additional consonants

The following set of consonants map mostly to the same phonemes.

Palatalisation

Palatalisation is a frequent feature of Kashmiri words. It is represented using ؠ [U+0620 ARABIC LETTER KASHMIRI YEH]. It mostly appears medially and finally: non-final/isolate forms have a small circle beneath them, eg. these are all the same character ؠ  ؠـ  ـؠـ  ـؠ.

Use preceding the inherent vowel seems to typically be transcribed using ê, eg. têmbar. At the end of a word, it is often transcribed using a superscript i, eg. tånⁱ.

Confusables

This section documents some common spelling errors. Most of the illustrations are from a Kashmiri Wikipedia page, but they do occur elsewhere, too.

Kashmiri doesn't use ي [U+064A ARABIC LETTER YEH], but people sometimes use it in Kashmiri text, failing to distinguish it from ی [U+06CC ARABIC LETTER FARSI YEH]. Here is an example from Wikipedia اِنسيكلؤپِيڈيا ạ̄insy̶k̸lu͑piy̶ɖy̶ạ̄. The correct spelling would be اِنسیکلۆپِیڈیا ạ̄insyklopiyɖyạ̄.

This can also lead to the use of characters such as ئ [U+0626 ARABIC LETTER YEH WITH HAMZA ABOVE], rather than ی + ٔ [U+06CC ARABIC LETTER FARSI YEH + U+0654 ARABIC HAMZA ABOVE].

That example also uses ك [U+0643 ARABIC LETTER KAF] instead of ک [U+06A9 ARABIC LETTER KEHEH], and ؤ [U+0624 ARABIC LETTER WAW WITH HAMZA ABOVE] instead of ۆ [U+06C6 ARABIC LETTER OE].

Non-final forms of ؠ [U+0620 ARABIC LETTER KASHMIRI YEH] may be incorrectly typed using ٮ + ۪ [U+066E ARABIC LETTER DOTLESS BEH + U+06EA ARABIC EMPTY CENTRE LOW STOP], eg. پٮ۪ٹھ pb̸˳ʈḣ in Wikipedia, rather than پؠٹھ pẏʈḣ.

(The same page has موسیٖقی, where the final ی [U+06CC ARABIC LETTER FARSI YEH] should perhaps be ؠ [U+0620 ARABIC LETTER KASHMIRI YEH] ?)

Consonant clusters & gemination

The diacritic ّ  [U+0651 ARABIC SHADDA] doubles the value of the consonant it is attached to. It is also used to explicitly signal an absence of intervening vowels between any two consonants, where needed.

Formatting characters

The Arabic script uses a large number of Unicode characters that affect the way that other characters are rendered. Many of those have no visible form of their own.

Modern Arabic-script text makes use of a relatively large set of invisible formatting characters, especially in plain text, many of which are used to manage text direction. For more details, see the Arabic overview.

Numbers, dates, currency, etc

I suspect that Kashmiri may use ٫ [U+066B ARABIC DECIMAL SEPARATOR] and ٬ [U+066C ARABIC THOUSANDS SEPARATOR], but need to confirm.
