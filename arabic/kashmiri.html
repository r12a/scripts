Kashmiri [arab] (draft)

Updated 16 April, 2021

This page gathers together basic information about the Arabic script and its use for the Kashmiri language. It aims (generally) to provide an overview of the orthography and typographic features, and (specifically) to advise how to write Kashmiri using Unicode.

See the Arabic page for most of the information about how the Arabic script works, and the orthography used for the Arabic language. This page aims to provide Kashmiri-specific information.

Phonetic transcriptions on this page should be treated as an approximate guide, only. Many are more phonemic than phonetic, and there may be variations depending on the source of the transcription.

Basic features

The Kashmiri Arabic orthography is derived from the Arabic/Persian abjads, where in normal use the script represents only consonant and long vowel sounds. However, Kashmiri is one of the Arabic orthographies that regularly indicates all vowel sounds, making it more like an alphabet.wkl,#Writing_system See the table to the right for a brief overview of features for the modern Kashmiri orthography using the Arabic script.

The Arabic alphabet has been adapted to cope with the many more vowels sounds in languages such as Kashmiri.

Kashmiri is principally written using the nasta'liq style of Arabic writing. Glyphs are more drawn out, and the baseline tends to be sloping from word to word.

Vowels

Vowel characters

Unusually among Arabic orthographies, and unlike Urdu, all vowel diacritics are normally shown when writing Kashmiri.

The set of characters needed to represent the Kashmiri vowels, including decompositions and compositions, is the following. In some cases, especially for joining forms, more than one of these characters need to be combined. For example, vowels in initial or isolated position are written with a preceding ا [U+0627 ARABIC LETTER ALEF]. For a list showing how such combinations map to vowel sounds see vowelOverview.

ٓ [U+0653 ARABIC MADDAH ABOVE] is only found separately in decomposed text. 

Observation: e, and its joining forms, is not particularly clear from the sources. Wikipedia originally used ێ [U+06CE ARABIC LETTER YEH WITH SMALL V] throughout, but now uses a combination of that and YEH BAREE, however it shows the initial form without ALEF. Daniels and LOC show a YEH with what looks like a combining breve above for the initial and medial forms, and ے [U+06D2 ARABIC LETTER YEH BARREE] with the same diacritic above for final and isolate.

Precomposed characters with hamza

The characters just above are all consonants followed by the vowel ə that Unicode represents in NFC-normalised text as single, precomposed characters. These are therefore included in the Kashmiri repertoire for representing those sound combinations.

Only ۂ [U+06C2 ARABIC LETTER HEH GOAL WITH HAMZA ABOVE] and ؤ [U+0624 ARABIC LETTER WAW WITH HAMZA ABOVE] decompose and recompose during normalisation. The other 3 neither decompose nor recompose during normalisation. This means that applications need to apply special rules to recognise the latter as equivalent, regardless of their composition.

Deprecated vowel character

The Unicode character ٳ [U+0673 ARABIC LETTER ALEF WITH WAVY HAMZA BELOW] represents a valid visual arrangement for Kashmiri, but is deprecated by the Standard, in favour of the decomposed sequence اٟ [U+0627 ARABIC LETTER ALEF + U+065F ARABIC WAVY HAMZA BELOW], because there is no normalisation equivalence.

Vowel absence

Kashmiri uses ْ [U+0652 ARABIC SUKUN] (jazm) to kill the vowel after a consonant.sle Two common alternative shapes are shown in fig_sukun.

Sometimes people use ۡ [U+06E1 ARABIC SMALL HIGH DOTLESS HEAD OF KHAH] for the shape on the right, but it's better to find a font that produces that glyph.

The following set of consonants map mostly to the same phonemes.

Palatalisation

Palatalisation is a frequent feature of Kashmiri words. It is represented using ؠ [U+0620 ARABIC LETTER KASHMIRI YEH]. It mostly appears medially and finally: non-final/isolate forms have a small circle beneath them, eg. (these are all the same character) ؠ    ؠـ    ـؠـ    ـؠ

Use preceding the inherent vowel is typically transcribed using ê, eg. têmbar. At the end of a word, it is often transcribed using a superscript i, eg. tånⁱ.

Gemination

The diacritic ّ  [U+0651 ARABIC SHADDA] doubles the value of the consonant it is attached to.

It is also used to explicitly signal an absence of intervening vowels between any two consonants, where needed. IS THIS TRUE?

Confusables

This section documents some common spelling errors. Most of the illustrations are from a Kashmiri Wikipedia page, but they do occur elsewhere, too.

Kashmiri doesn't use ي [U+064A ARABIC LETTER YEH], but people sometimes use it in Kashmiri text, failing to distinguish it from ی [U+06CC ARABIC LETTER FARSI YEH]. Here is an example from Wikipedia اِنسيكلؤپِيڈيا ạ̄insy̶k̸lu͑piy̶ɖy̶ạ̄.The correct spelling would be اِنسیکلۆپِیڈیا ạ̄insyklopiyɖyạ̄

This can also lead to the use of characters such as ئ [U+0626 ARABIC LETTER YEH WITH HAMZA ABOVE], rather than ی + ٔ [U+06CC ARABIC LETTER FARSI YEH + U+0654 ARABIC HAMZA ABOVE].

That example also uses ك [U+0643 ARABIC LETTER KAF] instead of ک [U+06A9 ARABIC LETTER KEHEH], and ؤ [U+0624 ARABIC LETTER WAW WITH HAMZA ABOVE] instead of ۆ [U+06C6 ARABIC LETTER OE].

Non-final forms of ؠ [U+0620 ARABIC LETTER KASHMIRI YEH] may be incorrectly typed using ٮ + ۪ [U+066E ARABIC LETTER DOTLESS BEH + U+06EA ARABIC EMPTY CENTRE LOW STOP], eg. پٮ۪ٹھ pb̸˳ʈḣ in Wikipedia, rather than پؠٹھ pẏʈḣ

(The same page has موسیٖقی where the final ی [U+06CC ARABIC LETTER FARSI YEH] should perhaps be ؠ [U+0620 ARABIC LETTER KASHMIRI YEH] ?)

Formatting characters

The Arabic script uses a large number of Unicode characters that affect the way that other characters are rendered. Many of those have no visible form of their own.

Modern Arabic-script text makes use of a relatively large set of invisible formatting characters, especially in plain text, many of which are used to manage text direction. For more details, see the Arabic overview.

Numbers, dates, currency, etc

I suspect that Kashmiri may use ٫ [U+066B ARABIC DECIMAL SEPARATOR] and ٬ [U+066C ARABIC THOUSANDS SEPARATOR], but need to confirm.

Text direction

Text is normally written horizontally, right to left, however numbers and non-Arabic script text runs left to right.

See the Arabic overview for more details, especially related to sequences of items and numbers.

Glyph shaping & positioning

This section brings together information about the following topics: writing styles; cursive text; context-based shaping; context-based positioning; baselines, line height, etc.; font styles; case & other character transforms.

You can experiment with examples using the Kashmiri character app.

Kashmiri written in the Arabic script is cursive, and there are combining characters and special joining behaviours.

The orthography has no case distinction, and no special transforms are needed to convert between characters.

See the Arabic overview for details.

Cursive script

Arabic script is always cursive, ie. letters in a word are joined up. Fonts need to produce the appropriate glyph for a letter, according to its visual context, but the code point used doesn't change. This results in four different shapes for most letters (including an isolated shape). Ligated forms also join with characters alongside them.

In the lists below 30 Kashmiri letters are dual-joining, whereas 17 join only to the right. However, the high frequency of the latter and short word lengths produce text that doesn't usually have long joined sequences (see fig_unjoined).

Cursive joining forms

Most dual-joining characters add or become a swash when they don't join to the left. A number of characters, however, undergo additional shape changes across the joining forms. fig_joining_forms and fig_right_joining_forms show the basic shapes in Kashmiri and what their joining forms look like.
