Cyrillic character notes

Contents

This page lists characters used in orthographies based on the Cyrillic script and provides information about them.

Character notes have companion pages that summarise how a script is used for a specific orthography. For the Cyrillic script, see Mongolian (Cyrillic), Russian, and Ukrainian.
