Hebrew character notes

Contents

This page lists characters used in orthographies based on the Hebrew script and provides information about them.

Notes on Biblical Hebrew only occur where there is something to say that is not already in the Modern Israeli Hebrew description.

Character notes have companion pages that summarise how a script is used for a specific orthography. For the Hebrew script, see Modern Israeli Hebrew.
