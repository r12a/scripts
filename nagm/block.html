Nag Mundari character notes

Contents

This page lists characters used in orthographies based on the Nag Mundari (Mundari Bani) script and provides information about them.

Character notes have companion pages that summarise how a script is used for a specific orthography. For the Bengali script, see Mundari (Nag Mundari).
