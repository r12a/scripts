Sunuwar character notes

Contents

This page lists characters used in orthographies based on the Sunuwar (Kõits Brese) script and provides information about them.

Character notes have companion pages that summarise how a script is used for a specific orthography. For the Sunuwar script, see Kiranti-Kõits (Sunuwar).

Sunuwar is scheduled for inclusion into the Unicode repertoire in version 16, and this page uses the code points that are expected to be used then. Because Sunuwar is not yet encoded in Unicode, some links may not produce results.
