Orthography notes switch

This page allows you to quickly find and compare sections across the orthography notes pages.


Select/set a fragment id:

The pulldown selection control lists sections that usually recur through all of the orthography notes pages. Options that begin with "—" point to sections that only appear for certain types of character.
You can also type the section id into the box on the right. This is particularly useful if you want to go to a section that is not common to many pages.
