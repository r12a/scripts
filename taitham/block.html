This page lists characters in the following Unicode block and provides information about them.

  • Tai Tham

This is not authoritative, peer-reviewed information – these are just notes I have gathered and copied from various places.

For a summary of the script and its use in writing systems, see the page Tai Tham script summary. For similar information related to other scripts, see this list.

We have usage data for no languages that use the Tai Tham script.

In order to allow the fonts to work on all platforms, rather than use '-' as a generic base for vowels and diacritics, this document uses the character WA, which is a simple circle.

Find

Read more about how to use this page.

To find a character by codepoint, type #char0000 at the end of the URL in the address bar, where 0000 is a four-figure, hex codepoint number, all in uppercase. Or type the character or the hex number in the Find control above.

To view this page as intended, you need a Tai Tham font. This page comes with a Tai Tham webfont. Click the blue vertical bar at the bottom right of the page to apply other fonts, if you have them on your system. For transcriptions I recommend the excellent and free Doulos SIL font. The large character in the box will not be rendered unless the webfont downloaded with the page or a system font has a glyph for it. (If there is no glyph and you want to see what it looks like, click on See in UniView.)

See in UniView

Repetition / disambiguation mark ʻ

Northern Thai

Used at the end of a word to indicate that it should be repeated, eg. ᨲ᩵ᩣ᩠ᨦ taŋ to be different, vs. ᨲ᩵ᩣ᩠ᨦ᩻ taŋtaŋ different in my view. e

Also used to indicate 'double-acting' consonants, stored where the consonant would be stored if there were a separate consonant used, eg. compare ᩮᨡᩢ᩶ᩣᨡᩬᨦ ekʰá²ākʰōṉ̇ kʰawkʰoŋ belongings with an alternative spelling using this diacritic, ᨡᩢ᩶᩻ᩬᨦ kʰá²ʻōṉ̇. e

A third use is somewhat different. A subjoined letter may be medial, final, or the start of a new syllable. Mai sam can be used to indicate that a consonant begins a new syllable, in which case it is stored following the subjoined form and indicates that it is the consonant that starts the new syllable, eg. compare ᨳᩫ᩠ᨶᩁ tʰö͓ṉrᵃ tʰonra with ᨳᩫ᩠ᨶ᩻ᩁ tʰö͓ṉʻr tʰanon path (final r is pronounced as n). e
