Tai Tham character notes

Updated Sun 21 Oct 2014 • tags bopomofo, chinese, scriptnotes

In this page I am exploring typing order when creating orthographic syllables in Tai Tham. It starts from work by Wyn Owen (p146), and builds on it.

The aim is to see whether there is an optimal order for input of characters, and to explore some anomalies. I start by mapping out what seems to work with a couple of fonts: Lamphun for Northern Thai, and A Tai Tham KH New for Khün.

The third column illustrates the pronunciation order, then shows the order accepted by the fonts, and the preferred order (still to be checked!). The top left cell gives an id, which can be used (in lowercase) to link to an example.
