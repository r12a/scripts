New Tai Lü character notes

Contents

This page lists characters used in orthographies based on the New Tai Lü script and provides information about them.

Character notes have companion pages that summarise how a script is used for a specific orthography. For the New Tai Lü script, see Tai Lü (New Tai Lü).
